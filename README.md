# Tasks-Project-App | مدير المهام اليومي 📝

Tasks-Project-App هو تطبيق ويب تفاعلي مصمم لمساعدتك على تنظيم حياتك اليومية، جدولة مهامك، وضمان عدم نسيان أي التزام بفضل نظام التنبيهات المدمج. التطبيق يركز على الإنتاجية والبساطة في الاستخدام.

## 🚀 نظرة عامة
تم بناء هذا التطبيق ليكون رفيقك الرقمي في إدارة المهام، حيث يوفر واجهة مستخدم بديهية تتيح لك إضافة، تعديل، وحذف مهامك اليومية بسهولة، مع دعم كامل للغة العربية.

## 🛠 التقنيات المستخدمة
تم تطوير المشروع باستخدام مجموعة من التقنيات الحديثة لضمان الأداء السريع والاستقرار:

*   **Frontend:** React.js, TypeScript.
*   **Styling:** Bootstrap, Custom CSS (تصميم عصري ومتجاوب).
*   **State Management:** React Hooks (لإدارة حالة المهام والتنبيهات).
*   **Data Persistence:** LocalStorage (لحفظ مهامك محلياً على المتصفح).
*   **Deployment:** GitHub Pages.

## 🌟 الميزات الرئيسية
*   **إدارة المهام:** إضافة مهام جديدة مع إمكانية تحديد أولويات لكل مهمة.
*   **التنبيهات:** نظام تنبيه ذكي لتذكيرك بالمهام العالقة.
*   **الواجهة العربية:** دعم كامل لاتجاه النص (RTL) لضمان تجربة مستخدم مريحة.
*   **التصميم المتجاوب:** التطبيق يعمل بكفاءة على كافة الشاشات (كمبيوتر، تابلت، وهواتف ذكية).
*   **حفظ البيانات:** بفضل تقنية LocalStorage، ستظل مهامك محفوظة حتى بعد إغلاق المتصفح.
